HTTP

HTTP står för HyperText Transfer Protocol transactions. Varje gång man går in på hemsidan så kommunicerar våran dator med HTTP med hjälp av speciell web software som gör att våran dator kan förstå HTTP. Genom HTTP kommunicerar den med serven som gör att vi får fram allt content. Men hjälp med HTTP kan vi överföra bilder, filmer, ljud, filer, data och mycket mer som brukar ingå i en hemsida.

TCP

TCP står för Transmission Control Protocol va den gör att om du vill skicka ett meddelande så delar den upp meddelandet i olika paket för inte överbelasta internet och skickar dem olika paketen till IP. Så alltså det är en protocol som gör att det finns en fungerande kommunikation mellan sändare och mottagare.

IP

IP står för Internet Protocol. IP tar emot pakten och skickar ett bekräftelse till sändaren att alla paket har kommit fram. Och sedan skickar vidare till TCP och sätter ihop i den ordning ska vara i.
