Element

Ett element används för att beskriva vad som finns på en hemsida. I dokumentet där du skapar din hemsida är dokumenten fyllt med html element. Element heter root element för innehåller alla element i dokumenten. Med html element är den indelad två delar head och body. Head element innehåller element som inte returnerar olika delar av dokument. T.ex titel, style, sheets, scripts och metadata. Body element är då alltså tvärt om den returnerar allt till användarna som dem kan se på dokumenten t.ex bilder, texter, knappar och mycket mer.

Tag

En tagg består av ett element som vanligtvis viss använder förkortning till elementens namn och tagarna använder angle brackers. t.ex < body >
< /body >. tag är alltså då byggstenen till dokumentet allt från text, bilder

Attribut

En attribut hjälper elements att förtydliga en tagg eller modifiera ett element. t.ex skriver man
< img> < /img> o inte < image> < /image> detta hjälper att det blir mindre kod i dokumentet.
